About

New Eden Trade Lab provides tools and utilities for traders and market-focused capsuleers in EVE Online. The goal of this project is to make pricing, order tracking, and day-to-day trade decisions easier for capsuleers. This is an independent community project and is not affiliated with or endorsed by CCP hf.
